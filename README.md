# Spring Boot Student CRUD API

A simple Spring Boot REST API project demonstrating CRUD operations.

## Features

* Create Student (POST)
* Get Students (GET)
* Update Student (PUT)
* Delete Student (DELETE)

## Technologies Used

* Java 21
* Spring Boot 4
* Maven/Gradle
* Postman

## Endpoints

### Get All Students

GET /students

### Create Student

POST /students

### Update Student

PUT /students/{name}

### Delete Student

DELETE /students/{name}
